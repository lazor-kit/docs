# Getting Started

Follow these steps to integrate the LazorKit wallet into your React Native application.

## 1. Wrap your App with `LazorKitProvider`

The `LazorKitProvider` manages the global state of the wallet connection. Wrap your root component with it.

```tsx
import React from 'react';
import { LazorKitProvider } from '@lazorkit/wallet-mobile-adapter';

export default function App() {
  return (
    <LazorKitProvider
      rpcUrl="https://api.devnet.solana.com"
      ipfsUrl="https://portal.lazor.sh"
      paymasterUrl="https://lazorkit-paymaster.onrender.com"
      isDebug={false} // Set to true for development logging
    >
      <YourMainComponent />
    </LazorKitProvider>
  );
}
```

## 2. Connect the Wallet

Use the `useLazorWallet` hook to access the `connect` function.

```tsx
import { useLazorWallet } from '@lazorkit/wallet-mobile-adapter';
import { Button, View, Text } from 'react-native';

export function WalletConnect() {
  const { connect, isConnected, isConnecting, wallet, error } = useLazorWallet();

  const handleConnect = async () => {
    try {
      await connect({
        redirectUrl: 'your-app-scheme://wallet-callback', // Must match your Expo scheme
        onSuccess: (walletInfo) => {
          console.log('Connected to:', walletInfo.smartWallet);
        },
        onFail: (err) => {
          console.error('Connection failed:', err);
        }
      });
    } catch (e) {
      console.error(e);
    }
  };

  if (isConnected && wallet) {
    return (
      <View>
        <Text>Connected: {wallet.smartWallet}</Text>
      </View>
    );
  }

  return (
    <Button 
      title={isConnecting ? "Connecting..." : "Connect Wallet"} 
      onPress={handleConnect} 
      disabled={isConnecting}
    />
  );
}
```

## 3. Sign a Message

Once connected, you can sign messages using the `signMessage` function.

```tsx
import { useLazorWallet } from '@lazorkit/wallet-mobile-adapter';

// ... inside your component
const { signMessage } = useLazorWallet();

const handleSign = async () => {
  const message = "SGVsbG8gV29ybGQ="; // "Hello World" in Base64

  await signMessage(message, {
    redirectUrl: 'your-app-scheme://sign-callback',
    onSuccess: (result) => {
      console.log('Signature:', result);
    }
  });
};
```
