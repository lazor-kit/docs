# useLazorWallet

The `useLazorWallet` hook is the primary interface for interacting with the wallet adapter.

```typescript
import { useLazorWallet } from '@lazorkit/wallet-mobile-adapter';
```

## Return Values

The hook returns an object with the following properties and methods:

### State

| Property | Type | Description |
|----------|------|-------------|
| `wallet` | `WalletInfo \| null` | The currently connected wallet information. |
| `isConnected` | `boolean` | `true` if a wallet is connected. |
| `isConnecting` | `boolean` | `true` while the connection flow is active. |
| `isSigning` | `boolean` | `true` while a signing operation is in progress. |
| `isLoading` | `boolean` | `true` during any async operation (connect, disconnect, sign). |
| `error` | `Error \| null` | The last error that occurred, if any. |
| `connection` | `Connection` | The underlying Solana `Connection` object. |

### Methods

#### `connect(options: ConnectOptions)`

Initiates the wallet connection flow. Opens the system browser for authentication.

- **options**: `ConnectOptions`
  - `redirectUrl` (required): The deep link URL to return to your app.
  - `onSuccess` (optional): Callback with `WalletInfo`.
  - `onFail` (optional): Callback with `Error`.

#### `disconnect(options?: DisconnectOptions)`

Disconnects the current wallet and clears the session.

- **options**: `DisconnectOptions` (optional)
  - `onSuccess` (optional): Callback on success.
  - `onFail` (optional): Callback on error.

#### `signMessage(action: string | SmartWalletActionArgs, options: SignOptions)`

Signs a message or executes a smart wallet action.

- **action**:
  - `string`: A Base64-encoded string to sign (Raw Message Signing).
  - `SmartWalletActionArgs`: An object defining a smart wallet transaction (Execute, CreateChunk, etc.).
- **options**: `SignOptions`
  - `redirectUrl` (required): The deep link URL to return to your app.
  - `onSuccess` (optional): Callback with the result (Signature or Transaction ID).
  - `onFail` (optional): Callback on error.
