# LazorKit React Native SDK

Native React Native adapter for LazorKit smart wallets.

## Why LazorKit React Native?
- **Native Experience**: Built for React Native & Expo
- **WebAuthn**: Secure, biometric authentication
- **Smart Wallets**: Advanced on-chain features
- **Gasless**: Built-in paymaster support
