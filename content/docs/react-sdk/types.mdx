# Type Definitions

## Core Types

### SignAndSendTransactionPayload

Payload structure for transaction signing.

```typescript
interface SignAndSendTransactionPayload {
  instructions: TransactionInstruction[];
  transactionOptions?: {
    feeToken?: string;
    addressLookupTableAccounts?: AddressLookupTableAccount[];
    computeUnitLimit?: number;
    clusterSimulation?: 'devnet' | 'mainnet';
  };
}
```

### WalletAccount

Connected wallet information.

```typescript
interface WalletAccount {
  publicKey: string;          // Passkey public key (base64)
  smartWallet: string;        // Smart wallet address (base58)
  isConnected: boolean;       // Always true when account exists
  isCreated: boolean;         // True if wallet was just created
}
```

### ConnectResponse

Passkey creation result.

```typescript
interface ConnectResponse {
  publicKey: string;              // Base64 encoded public key
  credentialId: string;           // WebAuthn credential ID
  isCreated: boolean;             // true = new passkey, false = existing
  connectionType: 'create' | 'get';
  timestamp: number;
}
```
